# CodeAlpha_Hangman_Game

## Description
A simple, text-based Hangman game developed as Task 1 for the CodeAlpha Python Programming Internship. The player tries to guess a hidden word one letter at a time within a limited number of attempts.

## Features & Scope
* **Predefined Word List:** The game randomly selects a word from a predefined list of 5 words (no external files or APIs used).
* **Attempt Limit:** Players are allowed a maximum of 6 incorrect guesses before the game ends.
* **Console-Based:** The game runs entirely in the terminal with basic text input and output (no GUI or audio).

## Key Concepts & Technologies Used
* **Language:** Python
* **Concepts:** `random` module, `while` loops, `if-else` conditions, `strings`, and `lists`.
